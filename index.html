Personalized Project Reference

Capture and paste your code segments below. These will be available to you on exam day only if submitted as final in the AP Digital Portfolio by the deadline. Screen captures must not be blurry and text must be at least 10-point font size. Do not include comments in your code segments.

Procedure

Two code segments containing a student-developed procedure that implements an algorithm and a call to that procedure. Built-in or existing procedures and language structures are not considered student-developed.

i Student-developed procedure that:
  • Defines the procedure's name and return type (if necessary)
  • Contains and uses one or more parameters that have an effect on the functionality of the procedure
  • Implements an algorithm that includes sequencing, selection, and iteration
ii Where your student-developed procedure is being called in your program.
List

Two code segments containing a list (or other collection type) being used to manage complexity. The data abstraction must manage complexity by making the program easier to develop or maintain.

i How data have been stored in the list.
ii The data in the same list being used — such as creating new data from the existing data or accessing multiple elements — as part of fulfilling the program's purpose.
