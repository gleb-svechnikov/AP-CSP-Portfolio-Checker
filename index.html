Digital Portfolio — Written Responses

Complete the six written response sections below. These are also submitted at digitalportfolio.collegeboard.org.

Written Responses — 6 rows, 6 points total
Row 1 (1 pt): Describe the overall purpose of the program. Describe what the video shows, including the program's input and output. Describe the functionality demonstrated in the video.
Row 2 (1 pt): Include two code segments: (i) showing the list being created/stored, and (ii) showing the list being used. Identify the name of the list and describe what data it represents.
Row 3 (1 pt): Explain how the list manages the complexity of the program. Describe how the program would be different without using this list.
Row 4 (1 pt): Include two code segments: (i) showing the procedure definition, and (ii) showing a call to the procedure. Identify the name, parameter(s), and how they affect the procedure's behavior.
Row 5 (1 pt): Describe what the identified algorithm does and how it achieves its purpose. Explain how it includes sequencing, selection, and iteration.
Row 6 (1 pt): Describe two calls to the procedure using different arguments. For each: the argument, condition tested, and result.
